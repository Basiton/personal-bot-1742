СОЗДАНИЕ И СВЯЗЫВАНИЕ
═══════════════════════════════════════════════════════════════

/showcase create +1 Название         → Создать витрину
/showcase link +1 @channel           → Привязать канал
/showcase unlink +1                  → Отвязать витрину

═══════════════════════════════════════════════════════════════
ПРОСМОТР
═══════════════════════════════════════════════════════════════

/showcase list                       → Все витрины
/showcase info +1                    → Информация о витрине

═══════════════════════════════════════════════════════════════
НАСТРОЙКА
═══════════════════════════════════════════════════════════════

/showcase set +1 avatar              → Аватар (затем отправить фото)
/showcase set +1 title "Новое"       → Название
/showcase set +1 about "Описание"    → Описание
/showcase set +1 info title:X|about:Y → Всё сразу ✨ НОВОЕ!
/showcase set +1 post "Текст"        → Пост
/showcase set +1 post_pin "Текст"    → Закреплённый пост

═══════════════════════════════════════════════════════════════
ПРИМЕРЫ
═══════════════════════════════════════════════════════════════

✅ Создать витрину Kelly's:
   /showcase create +1 Kelly's Showcase

✅ Обновить всё сразу:
   /showcase set +1 info title:Kelly's|about:Best content

✅ Создать пост:
   /showcase set +1 post "Новое поступление!"

✅ Посмотреть все витрины:
   /showcase list
